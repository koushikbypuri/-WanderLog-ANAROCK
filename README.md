# WanderLog

A React-based travel bucket list application that allows users to explore countries, view details, and maintain a travel bucket list.

## Features

* Authentication
* Protected Routes
* Session Persistence
* Country Search
* Region Filtering
* Country Details
* Bucket List
* Visited Countries

## Run Locally

```bash
npm install
npm run dev
```
